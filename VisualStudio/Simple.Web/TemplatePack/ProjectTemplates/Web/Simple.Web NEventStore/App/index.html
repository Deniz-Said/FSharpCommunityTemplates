Welcome!

Welcome to the Simple.Web and NEventStore F# project!

This (pure F#) project contains a few things to get started using Simple.Web, including the beginnings of a Twitter Bootstrap UI (Bootswatch's Yeti, to be precise), less the JavaScript. (This is so you can use Angular-UI or the raw Twitter Bootstrap JavaScript–your choice.) It also setup to use Font Awesome 4.0.3 out the gate. It is also configured to use NEventStore with a "quick and dirty" bus. See this blog post and this blog post for some examples of how to get started.

(This template does not use Razor. If you'd like to use it, add the Simple.Web.Razor NuGet package.)

If you like this template (or don't), you might be interested in some of the other community-supported F# web project templates.

  • F# ServiceStack Template
  • F# MVC5 Template
  • F# Nancy Template
  • Frank — resource-oriented wrapper library for working with the Web API

Happy Coding!
